Kenneth Auchenberg

Front-end engineer with a passion for the web platform.

I'm Kenneth. I believe the web is the future for applications, which is why I have specialized in the browser.

I often speak in public; in communities, at conferences and in companies. Mostly I talk about front-end development, and the web platform, but you'll also hear me talk about things like architecture, minimalism and design, because I'm curious about how we, human beings, interpret and interact with the world.

I'm here to make the web a better place - especially for us developers.

Open-source projects

  • dependo / grunt-dependo

    April 2013

    dependo is a visualization tool that draws a force directed graph of JavaScript dependencies which has been annotated with either CommonJS or AMD.

    Featured by DailyJS, Webification.

  • cardio

    January 2013

    Cardio is a web application health tool. It provides a snapshot (via a bunch of metrics) of how "healthy" your application is by measuring HTML/CSS/JS complexity.

    Still not finialized, but I want to be able to generate a matrix, and store it in Graphite, so I can see how the "health" of my web app changes over time.

  • backbone.advices

    September 2012

    I missed "filters" from Ruby on Rails in Backbone, so I experimented with the "advice" concept from aspect-oriented programming, and added it as a mixin for Backbone.

  • jquery.mentionsInput

    November 2011

    jquery.mentionsInput is a small, but awesome UI component that allows you to "@mention" someone in a text message, just like you are used to on Facebook or Twitter.

    Featured by Smashing Magazine, DailyJS, JavaScript Weekly, Zach Snow, DailyWebTrends, and many others.

  • CSS Reloader for Firefox and Google Chrome

    March 2010

    CSS Reloader is a browser extension, that allows you to reload all the CSS of any site without having to reload the page itself.

    My goal with CSS Reloader was to enable developers to become more productive, by simplifing their workflow.

    Featured by: Design Shack, CSS Player.
